Surgimento da Internet

(voltar)

A História da internet remonta à década de 1960, quando os pesquisadores começaram a desenvolver uma rede de computadores chamada ARPANET. A ARPANET permitiu que os pesquisadores compartilhassem informações entre universidades e instituições governamentais. Nos anos seguintes, a rede cresceu e evoluiu, permitindo que mais pessoas se conectassem e compartilhassem informações. Na década de 1990, a internet se tornou acessível ao público em geral, e o uso do World Wide Web explodiu, permitindo que as pessoas acessassem informações de qualquer lugar..

Como funciona o HTML

(voltar)

HTML(Hypertex Markup Language) é a linguagem padrão usada para criar páginas da web. Ele funciona como um conjunto de instruções que dizem ao navegador da web como exibir o conteúdo da página.

O HTML usa "tags" (etiquetas) para marcar elementos na página, como cabeçalhos parágrafos, imagens e links. As tags são escritas em códigos cercados por colchetes angulares "< >" e fecahadas com uma barra "/".

O conteúdo da página é organizado em uma hierarquia de elementos aninhados. Por exemplo, um cabeçalho pode estar dentro de um parágrafo que, por sua vez, está dentro de uma seção.

Além disso, o HTML também permite a inclusão de outros recursos, como scripts e folhas de estilo, para controlar a aparência e comportamentos da páginas. Esses recursos geralmente são referenciados em tags separadas dentro do cabeçalho da página.

Quando um navegador da web carrega um página HTML, ele interpreta as tags exibe conteúdo da página de acordo com as instruções fornecidas pelo código HTML.

O que são tags e quais os tipos.

(voltar)

Em HTML, as tags são elementos que fornecem informações sobre como o navegador deve exibir o conteúdo em uma página da web. As tags HTML são escritas em pares: uma tag de abertura (por exemplo, <'tag'>).

Existem muitos tipos diferentes de tag em HTML, mas algumas das mais comuns incluem:

  1. <'html'>: Essa tag indica o início do documento html.
  2. <'head'>: Essa tag contém informações sobre a página, como o título e as ,metatags.
  3. <'title'>: Essa tag define o título da página, que é exibido na barra de título do navegador.
  4. <'body'>: Essa tag contém todo o conteúdo visível da página.
  5. <'h1'> a <'h6'>: Essas tags são usadas para criar cabeçalhos com diferentes tamanhos e níveis de importância.
  6. <'p'>: ESsa tag é usada para criar parágrafos de texto.
  7. <'a'>: Essa tag é usada para criar links para outras páginas web.
  8. <'img'>: Essa tag é usada para exibir imagens na página.
  9. <'ol'> e <'ul'>: Essas são tags para lista ordenadas e não ordenadas, respectivamente.
